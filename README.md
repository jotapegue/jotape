# Portfólio de João Pedro Figueira Garcia

Site-portfólio profissional, moderno e responsivo, criado em HTML, CSS e JavaScript puro.

## Conteúdo

- Início
- Sobre
- Experiência
- Tecnologias
- Contato

## Como rodar

### Opção 1: abrir diretamente

Abra `index.html` no navegador.

### Opção 2: servidor local

Use qualquer servidor estático. Exemplo com Python:

```bash
python3 -m http.server 8000
```

Depois acesse `http://localhost:8000`.

## Estrutura

- `index.html` - conteúdo e metadados SEO
- `styles.css` - layout responsivo e visual
- `script.js` - interação de menu e navegação
- `og-image.svg` - imagem Open Graph para compartilhamento

## Observações

- O projeto não depende de framework ou build step.
- A navegação principal direciona para os perfis do GitHub e LinkedIn.
